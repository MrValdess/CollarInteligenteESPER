# Smart Pet Collar

Este repositorio contiene el trabajo final de la asignatura Ingeniería de Sistemas de la Información.

Consiste en una simulación de un collar inteligente para mascotas (perros y gatos) que mide constantes vitales, localizaciones y posibles problemas de salud.
Emplea código en ESPER EPL, patrones usando nITROGEN, Java y RabbitMQ.
